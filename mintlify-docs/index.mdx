## How it works

GRAFT.TODAY runs three core pipelines for each tenant:

<Steps>
  <Step title="Prospecting">
    The prospecting pipeline scrapes websites, audits AI presence via Google Gemini, identifies pain points, and generates personalised cold email drafts for approval.
  </Step>
  <Step title="Engagement">
    Once a prospect becomes a lead, an 8-stage Inngest workflow produces a full solution: lead profiling → PRD writing → design concepts → code generation → deployment → offer dispatch.
  </Step>
  <Step title="Chat">
    Each tenant gets a GraftBot iframe chatbot that captures leads, books appointments, and hands off to human support — all scoped to their knowledge base.
  </Step>
</Steps>

## Key technologies

| Layer | Technology |
|---|---|
| Framework | Next.js 16, React 19, TypeScript |
| Database | PostgreSQL via Prisma ORM |
| Auth | Clerk (multi-tenant organisations) |
| Events | Inngest (async workflows + crons) |
| Payments | Paddle (Merchant of Record) |
| AI | Google Gemini, Stitch SDK, Jules Labs |
| Deployment | Vercel + Render |
