# Feature Plans

This directory holds `feature-plan.md` files produced by the Gold Rush Feature Pipeline.

## Usage

When you say "Execute the Feature Workflow for [Feature Name]", the agent will:

1. Create a `feature-plan.md` here (or a feature-specific file like `feature-name-plan.md`)
2. List every file to create or edit
3. Wait for your approval before proceeding

## Naming

- Use kebab-case: `lead-scoring-plan.md`, `tenant-onboarding-plan.md`
- Or generic: `feature-plan.md` for the current feature in progress
